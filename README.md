## âš™ï¸ Installation

### 1. Clone this repository

```bash
git clone https://github.com/punkbibi/Chatroom.git
cd flask-socketio-chat
````

### 2. Install Python dependencies

We recommend using a virtual environment:

```bash
python -m venv venv
source venv/bin/activate       # On Windows: venv\Scripts\activate
pip install -r requirements.txt 
```

(Optional for production: `pip install eventlet`)

### 3. Install and configure Ngrok

You need ngrok installed to use the `forwarding_url.sh` script:

* Download ngrok from [ngrok.com](https://ngrok.com/)
* Set up your authentication token (required to use ngrok)

---

## â–¶ï¸ Running the App & Public Sharing

For ease of use, the provided `forwarding_url.sh` script starts Flask and creates a public ngrok tunnel simultaneously.

### 1. Execute the startup script

```bash
chmod +x forwarding_url.sh
./forwarding_url.sh
```

### 2. Share the Public URL

The script prints the public URL generated by ngrok.
Anyone can use it to access your live chat application over the internet.

**To Stop:** Press `Ctrl+C`. The script gracefully shuts down both Flask and ngrok.

### 3. Local Access (Alternative)

Run the app locally without ngrok:

```bash
flask --app server/app run --host=0.0.0.0 --port=5000
# Access at http://localhost:5000
```

## ğŸ§  How It Works

* When a user joins, they send their username via the Socket.IO `"join"` event.
* The server tracks users in `active_users` and maps sessions in `sid_to_user`.
* Messages are broadcast to all connected clients using the `"message"` event.
* Chat history is stored in each userâ€™s browser via `localStorage`.
* Server restarts clear all in-memory chat history.
